# docker-tools

> A docker image with various tools, built from `masm/archlinux`

It currently includes:
* lsyncd
* nmap
* openssh
* rsync

## Install

Pull this image with `docker pull masm/tools`.

## Usage

Run it with `docker run masm/tools <tool> <argument> ...`.
